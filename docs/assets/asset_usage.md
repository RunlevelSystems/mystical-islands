# 🧰 Asset Usage Standards
## Overview
Use assets to accelerate playable iteration, not to hide missing gameplay.

## Rules
- prioritize reusable modular kits
- maintain naming standards for prefabs/materials
- avoid one-off assets that break style consistency

## Production Priority
**P0 during MVP implementation.**
