# Artwork

This folder holds visual references for Mystical Isles. It is intended for concept art, loading screens, cutscene visuals, faction art, environment paintings, and promotional images.

---

## Planned Artwork Types

- concept art
- loading screen illustrations
- cutscene stills
- promotional key art
- faction banners and emblems
- environment mood pieces
- boss encounter illustrations
- item and relic showcase images

---

## Suggested File Areas

When images are added later, they can live in simple subfolders such as:

- [concept-art](concept-art/)
- [loading-screens](loading-screens/)
- [cutscenes](cutscenes/)
- [promotional](promotional/)
- [factions](factions/)
- [environments](environments/)

## Notes

- Keep image naming consistent.
- Use filenames that match the region, faction, or event they depict.
- Prefer high-level compositions that communicate mood quickly.
- Add alt text and captions when images are embedded in documentation.
