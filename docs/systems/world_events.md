# 📜 World Events System
## Overview
World events are timed or triggered scenarios that alter region conditions, risks, and opportunities.

## Gameplay Purpose
Keep the world reactive and reinforce cooperative play.

## Player Experience Goals
- dynamic world pacing
- social urgency and coordination
- memorable region-level moments

## Technical Notes
Event architecture should support modular triggers, rewards, and cooldown controls.

## Implementation Notes
MVP includes only small local events (e.g., ward instability, caravan defense).

## Examples
- temporary corruption bloom spawns elite enemies
- faction convoy event boosts merchant stock if defended

## Production Priority
**P1 local events, P3 world-scale chains.**
